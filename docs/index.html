a slightly stripped-down fontconfig is running in your browser using WebAssembly!

this is useful if you need to do your own font selection, as an alternate to font-manager, etc

you do have to add each ttf manually

check off some of the included demo fonts to add them to the fontconfig database for querying

or you can drag a file to the browser to add it to the database

then test out querying fonts you've loaded by filling out any or none of the different fields
